# Plan de Implementación: Estructura y Estilo Base

Este plan detalla las tareas para completar la Fase 1 del roadmap: crear la estructura HTML y el estilo CSS base del proyecto.

## Grupo de Tareas 1: Estructura HTML

1.  **Crear Archivo Principal:** Generar el archivo `punto-limpio.html` dentro del directorio `punto-limpio/`.
2.  **Definir Encabezado y Pie de Página:** Crear un `<header>` con un logo y una navegación básica, y un `<footer>` con la información de copyright.
3.  **Estructurar Secciones Principales:** Añadir las secciones semánticas (`<section>`) con IDs correspondientes a las funcionalidades del roadmap: `inicio`, `como-reciclar`, `mapa`, y `guia-separacion`.

## Grupo de Tareas 2: Estilo CSS Base

1.  **Crear Hoja de Estilos:** Generar el archivo `style.css` en el directorio `punto-limpio/`.
2.  **Definir Paleta y Tipografía:** Establecer las variables de color y las fuentes principales (`Nunito`) que se alineen con la estética "eco-friendly".
3.  **Aplicar Estilos Generales:** Dar estilo al `body`, `header`, `footer` y a los contenedores principales para crear un layout coherente y limpio.

## Grupo de Tareas 3: Diseño Adaptable (Responsive)

1.  **Crear Hoja de Estilos Responsive:** Generar el archivo `responsive.css`.
2.  **Añadir Media Queries:** Implementar reglas básicas para que la estructura se adapte correctamente a pantallas de dispositivos móviles.
