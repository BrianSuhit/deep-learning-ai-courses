# Punto-Limpio

## Input from stakeholders

**Engineering:** Necesitamos una landing page rápida, ligera y fácil de mantener. Priorizar el uso de HTML/CSS limpio y JavaScript básico para asegurar compatibilidad total en cualquier navegador.

**Product:** El objetivo es facilitar a los ciudadanos la ubicación de puntos de reciclaje. Debe incluir una sección clara de "Cómo reciclar", un mapa visual con los puntos y una guía de separación de residuos.

**Marketing:** El diseño debe ser amigable, intuitivo y visualmente limpio (estética "eco-friendly"). Debe inspirar confianza para que la gente realmente use los puntos de reciclaje en la ciudad.
