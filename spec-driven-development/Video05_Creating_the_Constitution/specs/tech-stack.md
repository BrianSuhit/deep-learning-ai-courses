# Technology Stack

To build a reliable and modern web application, we will use a popular stack centered around TypeScript, as requested by our engineering stakeholder.

- **Language**: **TypeScript** will be used for both the frontend and backend to ensure type safety and maintainability.
- **Web Framework**: We recommend **Next.js**, a popular React framework. It enables server-side rendering for a fast user experience, works well in modern browsers, and has excellent TypeScript support.
- **Styling**: To create an attractive and modern user interface, we will use **Tailwind CSS**, a utility-first CSS framework that allows for rapid UI development.
- **Dashboard & UI Components**: We will utilize a component library like **Shadcn/ui** to build the dashboard for agents and staff, ensuring easy and reliable access to features.
