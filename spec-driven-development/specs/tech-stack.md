# Tech Stack: Punto Limpio

Para cumplir con el requisito de una landing page rápida, ligera y de fácil mantenimiento, se utilizará un stack tecnológico simple y robusto.

## Frontend

*   **HTML5:** Para la estructura semántica del contenido.
*   **CSS3:** Para el diseño y la estética visual, manteniendo un estilo limpio y "eco-friendly". Se escribirá CSS puro para evitar dependencias.
*   **JavaScript (Vanilla):** Se usará JavaScript básico para funcionalidades interactivas, como el mapa, asegurando la máxima compatibilidad con todos los navegadores modernos sin necesidad de frameworks pesados.

## Alojamiento

*   La página será estática y se podrá alojar en cualquier proveedor de hosting simple (como GitHub Pages, Netlify, Vercel) para garantizar velocidad y bajo costo.

No se requiere un backend complejo en la fase inicial.
